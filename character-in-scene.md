# Character in Scene — Character Workflow Experience

One of the most important features inside the Video Creation Workspace is the **"Character in Scene"** system.

This feature is responsible for:

- Character creation
- Character consistency
- Scene-level character management
- Character styling
- Character emotions
- Character animation
- Multi-scene continuity

The experience should feel cinematic, intelligent, and extremely interactive.

---

## Entry Point — Character Section

Inside the Video Creation Workspace, when the user clicks on:

- Character Section
- Character Layer
- "Add Character" button
- Character thumbnail
- Scene Character Settings

A premium cinematic modal popup should appear.

---

## Character Selection Popup

**Popup Type:**
- Center modal popup
- Glassmorphism UI
- Background blur
- Cinematic dark overlay
- Smooth scale animation
- Soft glowing borders

**Popup Size:** 980px–1200px wide, auto height with internal scrolling if needed

**Spacing:** 32px outer padding · 24px internal section gaps · 16px card spacing · Balanced whitespace

**Border Radius:** 28px

**Background:** Matte black + gradient blur · Transparent layered effect

---

## Popup Header

**Top Area Includes:**
- Title: *"Create Your Character"*
- Subtitle: *"Generate cinematic characters for your scenes using AI."*

**Right Side:**
- Close button
- Help icon
- Recently used characters shortcut

**Spacing:** 12px between title and subtitle · 24px bottom spacing before content

---

## Main Character Options Grid

Show **4 large interactive option cards** in a 2×2 responsive grid with 24px gaps.

Each card includes:
- Large preview visual
- Icon
- Heading
- Short description
- Hover animation
- Glow border on hover
- Smooth transition
- Height: 260px–320px

## Character in Scene Experience

After character selection, the character automatically appears inside the scene.

**User Controls:**
- Drag character in scene
- Resize & reposition
- Replace outfit
- Change expression / emotion
- Change camera focus

**AI Automatically:**
- Matches scene lighting
- Matches shadows
- Matches environment colors
- Matches camera angle
- Maintains face consistency

---

## Character Controls Inside Scene

**Toolbar Options:**
